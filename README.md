## Instalar 💿

---

### `Requisitos`

- [Node](https://ubunlog.com/php-8-0-instalar-lenguaje-en-ubuntu/) v16.17.1
- [React](https://reactjs.org/) v18.2.0

Tras clonar este repositorio con 'git clone' (por ejemplo), ejecutar el siguiente comando:

```
npm install
```
